# Notes — Karpathy, "Software 3.0" talk (June 2025)

> Software 1.0 is code we write. Software 2.0 is weights we train.
> Software 3.0 is prompts we write in English — the LLM is the new
> computer, and natural language is the programming surface.

> Partial autonomy is a slider, not a switch: the same model ranges
> from autocomplete to fully agentic depending on how much of the
> loop you hand it.

My note: the "slider" framing is the part that stuck. It reframes
"how much should I let the AI do?" as a dial I set per task, not a
yes/no I decide once. Feeds [[records/themes/ai-as-co-thinker]].

Watched it as [the YC AI Startup School keynote](https://www.youtube.com/watch?v=LCEmiRjPEtQ).
