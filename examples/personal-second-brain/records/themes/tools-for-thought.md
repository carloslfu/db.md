# Tools for thought

How a note-taking system becomes a thinking system — not a place to
store what I already know, but an instrument that produces ideas I
wouldn't have had otherwise. This is the theme that justifies the
whole garden's existence.

## The core moves

- **Atomic notes** (Matuschak). One note, one idea, so notes can be
  recombined without dragging unrelated context along.
- **Write in your own words** (Ahrens). If I can't state it clearly in
  a full sentence, I haven't understood it. The writing *is* the
  thinking.
- **Link by context-of-use, not by topic** (both). The slip-box isn't
  a library sorted by subject; it's an associative web. The surprising
  link is the raw material for the next piece of writing.
- **Title as a claim, not a folder name** (Matuschak). A note titled
  with a claim I can agree or disagree with stays reusable; a note
  titled "Focus" dies in a drawer.

## My open questions

- Where's the line between a note system and a database? This garden
  is files + frontmatter + links — at what point did it quietly become
  the latter?
- How much pruning is healthy? I keep adding; I rarely garden the old
  notes. Is the slip-box supposed to be tended, or just fed?
